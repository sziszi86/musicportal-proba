# MusicPortal

A modern music web application built with Next.js 15, React 19, and TypeScript.

## Features

- Modern responsive design
- Album browsing and reviews
- Music player interface
- Admin dashboard
- Genre filtering

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- shadcn/ui
- Redux Toolkit
- Zod

## Installation

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
